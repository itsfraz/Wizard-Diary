## ✨ Features

- **📖 Magical UI**: A beautifully crafted ancient diary interface with leather binding, parchment textures, and magical glows.
- **🪄 Spellbinding Animations**: Smooth vanishing text, typing effects, and ambient glows powered by `framer-motion`.
- **⚡ Lightning Fast AI**: Integrated with the **Groq API** (`llama-3.1-8b-instant`) for incredibly fast and highly accurate responses.
- **🌗 Dual Modes**: Seamlessly toggle between the immersive "Book" view and a focused "Single" minimalistic view.

## 🚀 Getting Started

Follow these steps to cast the spell and run the app locally!

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/itsfraz/Wizard-Diary.git
cd "Wizard-Diary"
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Configure the Magic (Environment Variables)
You will need a Groq API Key to awaken the wizard.
1. Create a `.env` file in the root directory.
2. Add your Groq API key:
```env
VITE_GROQ_API_KEY=gsk_your_groq_api_key_here
```
> **Note:** Do not commit your `.env` file! A `.env.example` template is provided for reference.

### 4️⃣ Ignite the Runes
Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173/](http://localhost:5173/) in your browser to interact with the diary.
